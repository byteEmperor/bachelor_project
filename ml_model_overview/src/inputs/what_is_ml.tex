% Preamble
Machine Learning is a way to learn a function from data instead of writing said
function explicitly. With the aim of the project in mind, a good example would be
that instead of defining force fields, energy terms, and sampling procedures, you
would analyze thousands of protein-ligand complexes and have the model learn the
patterns. So the model infers the statistical structure.
